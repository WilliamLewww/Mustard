# Thane

### Getting Started:
- Extract Content
- Run main.exe

## Controls
### Basic
- Left arrow & Right arrow = Turn

### Slides
- Turn Keys + A = Regular Slide
- Turn Keys + S = Technical Slide
- Turn Keys + D = "Coleman Slide"

### Display
- O/P = turn on/off road lines
- K/L = turn on/off GUI
- I = reset thane/brake lines
