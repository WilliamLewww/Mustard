- Run main.exe

### Latest Update:
- Updated GUI
- Changed Default Board

## Controls
### Basic
- Spacebar = Push/Tuck
- Left arrow & Right arrow = Turn

### Slides
- Turn Keys + A = Regular Slide
- Turn Keys + S = Technical Slide
- Turn Keys + D = Shutdown Slide

### Display
- O/P = turn on/off road lines
- K/L = turn on/off GUI
- I = reset thane/brake lines
